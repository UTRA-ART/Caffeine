# Miscallaneous #

This folder contains miscallaneous packages for Caffeine. 

This folder contains the following packages: teleop_twist_keyboard, twist_mux & load_waypoints.

## Usage ## 
### teleop_twist_keyboard ### 

Please see README.md in the teleop_twist_keyboard folder for detailed usage. 
For most practical cases, the following command should be used:
```
roslaunch teleop_twist_keyboard keyboard_teleop.launch
```
This creates a node to allow for teleop control of Caffeine.

### twist_mux ### 
This package is automatically run when spawning caffeine through the description package. Therefore, no extra work is required to run this package.
